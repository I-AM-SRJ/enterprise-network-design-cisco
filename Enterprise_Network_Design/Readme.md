\# Enterprise Network Design (Cisco Packet Tracer)



\## Overview

This project demonstrates the design and implementation of a multi-site enterprise network connecting London, Leeds, Liverpool, and Leicester.


\## Technologies Used

\- VLANs (Network segmentation)

\- OSPF (Dynamic routing)

\- DHCP (Automatic IP assignment)

\- NAT (Internet access via London site)

\- ACLs (Security and traffic control)



\## Key Features

\- Inter-VLAN routing using router-on-a-stick

\- Dynamic routing using OSPF

\- Controlled internet access (London only)

\- Department-based access control (Employees, Executive, Finance)

\- Secure device configuration using SSH


\## How to Run

1\. Open the `.pkt` file in Cisco Packet Tracer

2\. Start simulation mode

3\. Test connectivity using ping
